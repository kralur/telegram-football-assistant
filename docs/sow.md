# Statement of Work (SOW)

## Project Background

This project is based on the development of a Telegram-based football assistant bot designed to provide users with fast and convenient access to football-related information.  
The application integrates external football data APIs and uses AI-powered natural language processing to improve user interaction and response clarity.

---

## Project Goals and Objectives

The main goal of this project is to design and implement a functional Telegram bot that delivers football data through natural language interaction.

The objectives of the project include:
- Developing a Telegram bot capable of handling user messages
- Integrating an external football data API
- Implementing AI-based natural language processing for user queries
- Storing user preferences and settings
- Providing personalized football updates

---

## Intended Users

The intended users of the system are football fans who want quick access to football information using the Telegram messenger.

Users are expected to:
- Interact with the bot using natural language queries
- Select favorite football teams
- Receive personalized notifications and updates

---

## Scope of Work

### Functional Requirements

The system shall:
- Accept natural language input from users
- Retrieve football data from an external API
- Generate AI-based text responses
- Display match schedules and results
- Display league standings
- Display player statistics
- Store user preferences in a database
- Send notifications related to selected teams

---

### Development Tools and Frameworks

The project will use the following tools and technologies:
- Python programming language
- Aiogram framework for Telegram Bot API
- External football data API
- PostgreSQL database
- AI language model API
- Git and GitHub for version control

---

### Assumptions and Constraints

Assumptions:
- Users have access to the Telegram messenger
- External football data APIs are available during development

Constraints:
- The project is developed by a single student
- The scope is limited to backend and bot functionality
- No machine learning model training is included

---

## Deliverables

The project deliverables include:
- Source code of the Telegram bot
- Configured GitHub repository
- Project documentation (proposal and SOW)
- Demo video demonstrating system functionality

---

## Acceptance Criteria

The project is considered complete if:
- The Telegram bot responds correctly to user queries
- Football data is retrieved and displayed accurately
- AI-generated responses are understandable and relevant
- User preferences are stored and applied
- The project is successfully demonstrated
