# Telegram Football Assistant Bot

## Project Overview
This project is a Telegram-based football assistant that provides users with match schedules, results, league standings, and player statistics using external football APIs.  
The bot also uses AI-powered natural language processing to generate user-friendly responses.

## Key Features
- Natural language interaction
- Match schedules and results
- League standings
- Player statistics
- Personalized notifications
- AI-generated text responses

## Technologies
- Python
- Telegram Bot API (Aiogram)
- Football API
- PostgreSQL
- AI (LLM-based text generation)

## Project Status
In development.
